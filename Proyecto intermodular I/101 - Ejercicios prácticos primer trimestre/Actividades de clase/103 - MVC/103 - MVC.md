En esta actividad se trabaja el concepto de **generación de contenido dinámico en la web** utilizando el microframework **Flask** y el formato de datos **JSON**. Flask es una herramienta muy versátil escrita en Python que permite crear aplicaciones web de forma rápida y sencilla, gestionando rutas, plantillas y respuestas al navegador.

El propósito de esta práctica es construir un **sitio web que muestre un Curriculum Vitae** a partir de datos almacenados en un archivo JSON. Con este enfoque se introduce el uso de **plantillas dinámicas** mediante el motor de plantillas **Jinja2**, incluido en Flask, que permite insertar variables dentro del código HTML.

Este ejercicio combina conocimientos de programación backend en Python, estructuración de datos con JSON, y diseño básico en HTML y CSS, sentando las bases para desarrollar aplicaciones web que separan los datos de su presentación visual.

---

Nos aseguramos de tener Flask instalado.

```
pip install flask
```

En el archivo JSON contiene los datos personales del individuo.

```
{
  "datos personales": {
    "nombre": "Ana",
    "apellidos": "Sánchez Suárez",
    "email": "info@ana.com",
    "telefono": "722289695",
    "direccion": "La calle de Ana",
    "codigo_postal": "46020 Valencia"
  },
  "experiencia": "",
  "formacion": ""
}
```

El formato **JSON** es ideal para almacenar información estructurada, ya que es fácilmente legible por humanos y por programas. En este caso, los datos se agrupan dentro de un objeto `datos personales`.


En el archivo HTML se define la estructura visual del sitio web utiliza **etiquetas Jinja2** para mostrar los datos que Flask enviará dinámicamente.

```
<!doctype html>
<html lang="es">
  <head>
    <title>Curriculum</title>
    <meta charset="utf-8">
    <style>
      html{background:grey;font-family:sans-serif;}
      body{background:white;margin:auto;min-height:200px;display:flex;width:600px;}
      #izquierda{flex:1;background:darkblue;padding:20px;}
      #derecha{flex:4;background:white;padding:20px;}
    </style>
  </head>
  <body>
    <div id="izquierda"></div>
    <div id="derecha">
      <h1>{{nombre}} {{apellidos}}</h1>
      <p>{{email}}</p>
      <ul>
        <li>{{telefono}}</li>
        <li>{{direccion}}</li>
        <li>{{codigo_postal}}</li>
      </ul>
    </div>
  </body>
</html>
```

Gracias al uso de las llaves, Flask reemplazará estos marcadores por los valores reales provenientes del archivo JSON.


En el archico de Python, el código gestiona la aplicación Flask, carga los datos del archivo JSON y renderiza el HTML con las variables correspondientes.

```
from flask import Flask, render_template_string
import json

app = Flask(__name__)

@app.route("/")
def cv():
    # Abrimos el archivo JSON y cargamos los datos
    with open("curriculum.json", encoding="utf-8") as f:
        data = json.load(f)
    dp = data["datos personales"]

    # Leemos el archivo HTML
    with open("curriculum.html", encoding="utf-8") as f:
        html = f.read()

    # Renderizamos el HTML sustituyendo las variables con los datos del JSON
    return render_template_string(html, **dp)

# Iniciamos el servidor Flask
if __name__ == "__main__":
    app.run(debug=True)
```

**Ejecución y visualización.**

Abrimos un terminal en la carpeta del proyecto. Y ejecutamos lo siguiente:

```
python curriculum.py
```

Abrimos el navegador y entramos a:

```
http://127.0.0.1:5000/
```

Veremos el Curriculum Vitae de Ana Sánchez Suárez.

En definitiva, este proyecto representa un paso importante en la comprensión de cómo se construyen **aplicaciones web dinámicas**, sirviendo como base para desarrollar sitios más avanzados y personalizados en el futuro.
