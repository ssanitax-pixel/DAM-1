Después de crear el archivo, dentro escribimos el `DOCTYPE` para indicar que estamos utilizando HTML5.

```
<!doctype>
```

Definimos la etiqueta `<html>`, con su atributo para saber que idioma tiene el documento, importante también cerrarlo.

```
    <html lang="es">
    </html>
```

Creamos dentro de `<html>` la etiqueta `<head>`, que es donde irán los metadatos y otros elementos que no son visibles en la página, pero son importantes para su funcionamiento.

```
        <head>
        </head>
```
