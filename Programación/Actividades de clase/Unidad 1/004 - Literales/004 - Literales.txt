Vamos a crear un programa, en la que usaremos dos variables, del nombre (cadena) y edad (numérico) de un amigo o amiga, y finalmente haremos una salida en pantalla con un 'print'.

---

Primero declararemos las variables, aquí está un ejemplo:

```
nombre_amigo = "Alba"
```

Cuando ya tengamos las variables definidas, realizaremos un print para mostrar el mensaje en pantalla.

```
print("Mi amiga",nombre_amigo,"tiene",edad_amigo,"años")
```

Con este ejercicio practicamos cómo declarar variables con literales correctos (una cadena para el nombre y un número entero para la edad), y cómo mostrar un mensaje que combina texto y variables usando la función 'print'. Así reforzamos conceptos básicos y útiles para cualquier programa que maneje datos de personas.
