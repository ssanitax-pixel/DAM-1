MongoDB

Es una base de datos NoSQL
Se ha convertido en NOSQL

No SQL = No es SQL
NOSQL = Not Only SQL

Vamos hacia una fusión de las dos tecnologías

Las bases de datos SQL son buenas guardando tablas
Las bases de datos NoSQL (MongoDB) son buenas guardando documentos

Yo tengo un listado de clientes?
Todo aquello cuyo espacion natural es una excel, se suele guardar en SQL

Yo tengo un listado de documentos?
Todo aquelllo cuyo espacio natural es un documento, se suele guardar en MongoDB

Yo tengo un contunto de conceptos y lo que busco es similitud entre ellos?
Se suele usar ChromaDB (DB vectoriales)
